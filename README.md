# PHYTON
Proyecto: DataProject: Lógica. Katas Python

1.	DESCRIPCIÓN DEL PROYECTO

Este proyecto recoge la resolución de una serie de ejercicios prácticos de Python realizados como parte del curso. El objetivo principal ha sido afianzar los conceptos básicos del lenguaje, siguiendo estrictamente el contenido y las estructuras explicadas en el material del curso.

-	A lo largo del proyecto se han trabajado temas como:
-	Variables y tipos de datos
-	Estructuras de control (if, elif, else)
-	Bucles
-	Funciones y funciones lambda
-	Uso de map(), filter() y reduce()
-	Manejo de listas, tuplas, diccionarios y conjuntos
-	Programación orientada a objetos
-	Programas interactivos con input()

2.	PASOS SEGUIDOS DURANTE EL EJERCICIO

-	Análisis del enunciado
Cada ejercicio se ha analizado 7para entender qué se pedía exactamente y qué conceptos del curso debían aplicarse.
-	Diseño de la solución
El código se ha escrito siguiendo buenas prácticas básicas:
-	Uso de comentarios explicativos (#) dentro del código
-	Uso de funciones cuando tenía sentido
-	Pruebas y validación: Cada ejercicio se ha probado con distintos ejemplos para comprobar que el resultado era el esperado y que el programa se comportaba correctamente ante distintos casos.

3.	RESOLUCIÓN DE DUDAS

Durante el desarrollo del proyecto, en caso de dudas con algún ejercicio o concepto:
-	Se ha consultado a personas del entorno para contrastar ideas o enfoques.
-	También se ha utilizado ChatGPT como apoyo para: Aclarar conceptos teóricos, Verificar que las soluciones estuvieran dentro del temario,	Corregir errores y mejorar la comprensión de los ejercicios

Estas consultas se han usado siempre como herramienta de apoyo al aprendizaje, no como sustitución del trabajo personal.

4.	CONCLUSIÓN

Este proyecto ha servido para consolidar los conocimientos básicos de Python y ganar confianza en la resolución de problemas mediante programación. El proceso ha sido progresivo y enfocado en la comprensión real de cada ejercicio, sentando una buena base para continuar aprendiendo el lenguaje.
